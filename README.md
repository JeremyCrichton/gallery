# Gallery

## Description

A web app that allows users to sign up, upload, and manage photos, using modern technologies and services as outlined below.

## Tech Used and Services Integrated

- Next.js and React
- TypeScript
- Tailwind: Styles
- Shadcn/ui: Open source component lib
- Drizzle: ORM
- Vercel: Build, host, postgres db
- Clerk: Auth
- Posthog: Analytics
- Upstash: KV store for ratelimiting
- Sentry: Error management
- Uploadthing: File upload service for Next.js
